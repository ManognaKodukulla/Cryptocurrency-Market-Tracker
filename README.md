A responsive React application that displays real-time cryptocurrency prices and market statistics using the CoinGecko API.
</p>

---

## ✨ Features

- 📈 View real-time cryptocurrency market data
- 🔍 Search cryptocurrencies instantly
- 💰 Live price updates
- 📊 Display market capitalization
- 📉 View 24-hour price change
- 📱 Fully responsive user interface
- 🌐 CoinGecko API integration

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- Axios
- CoinGecko API
- Vercel

## ⚙️ Installation

```bash
git clone https://github.com/ManognaKodukulla/Cryptocurrency-Market-Tracker.git

cd Cryptocurrency-Market-Tracker

npm install

npm start
```
